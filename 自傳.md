## 📌 自傳

### 💡 對資訊領域的啟蒙

我對資訊科技的興趣可以追溯到國中時期。當時開始接觸電腦與數位世界，逐漸對「程式如何運作」產生好奇。我很好奇，人們是如何透過簡單的 **0 與 1 的邏輯運算，以及各種程式語言的設計，創造出一個完整的數位世界**。  
這份好奇心也成為我持續探索資訊領域的重要動力。

---

### 🎓 大學學習與技術基礎

進入資訊管理系後，我開始系統性地學習程式設計與資訊相關知識。在校期間我以 **Python** 作為主要學習語言，練習撰寫各種程式來解決問題，例如資料處理、簡單自動化工具，以及程式邏輯的實作。

同時我也接觸了 **網頁前端技術**，包含 **HTML、CSS 與 JavaScript**。透過實作網頁頁面，我學習如何進行版面切版、製作基本互動效果，以及思考使用者在瀏覽網頁時的體驗。這些經驗讓我逐漸理解前端開發不只是讓畫面呈現出來，也需要兼顧結構、可讀性與使用便利性。

---

### 🛠 學習方式與做事態度

在學習與實作過程中，我養成了先 **釐清需求與問題，再逐步拆解流程** 的習慣。當遇到問題時，我會嘗試將複雜的問題拆成小步驟逐一驗證，並透過查詢文件或測試程式來找出解決方式。

在開發過程中，我也會使用 **Git 與 GitHub 進行版本控制**，讓程式修改過程可以被記錄與追蹤，同時也能整理自己的學習成果並分享在 GitHub 上，作為持續累積的作品與學習紀錄。
